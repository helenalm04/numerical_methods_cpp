# Numerical Methods in C++

C++ implementations of numerical differentiation, interpolation, numerical integration, and trajectory simulation.

## Build
cmake -S . -B build
cmake --build build

## Run
./build/q2
./build/q3
./build/q4 25.0

## Report
See `docs/Report.pdf`.

## Data
Coursework datasets may be restricted. Place local data files in `data/` (see `data/README.md`).
